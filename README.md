### Что работает:
+ Камера(Фото, Видеозапись) фронтальная, задняя.
+ Фонарик
+ Сенсоры (accelerometer, proximity, light)
+ Аудио
+ Видео
+ RIL(USSD, 2G, 3G)
+ FM-Radio
+ Wi-Fi, Bluetooth, GPS (+GNSSV)
+ DM-Verity(По умолчанию отключен)
+ Устройство сертифицированно в Google Play и проходит SafetyNet
+ SELinux Encforcing
